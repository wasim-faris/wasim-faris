### 🔄 SkillSwapper
> *Peer-to-peer skill exchange platform*

A platform where users swap skills with each other — if you know Django and your friend knows JavaScript, you teach each other. No money. Just knowledge.

**Built with:**
`Django` `React` `JWT Auth` `REST API` `PostgreSQL` `Tailwind CSS` `Framer Motion`

**Key Features:**
- Skill matching algorithm
- User profiles with skill listings
- Real-time swap requests
- Glassmorphism UI with cinematic animations

### 🛍️ no-co
> *Production-grade e-commerce platform*

A fully functional e-commerce platform built to H&M-level standards — designed in Figma, database-modelled before a single line of code was written.

**Built with:**
`Django` `React` `Razorpay` `PostgreSQL` `Figma` `Tailwind CSS`

- Full cart, wishlist & order system
- Admin panel with product management
- Variant products (size, color)
- Coupon & offer system
- Referral system
- Forgot password / reset flow
- Razorpay payment integration
